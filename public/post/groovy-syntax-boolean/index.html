Groovy 语法 - Boolean

2019年6月28日
| 阅读

Boolean 是一种特殊的数据类型，用于表示真值：true 和 false。将此数据类型用于跟踪真/假条件的简单标志。

布尔值可以存储在变量中，分配到字段中，就像任何其他数据类型一样：

def myBooleanVariable = true
boolean untypedBooleanVar = false
booleanField = true

true 和 false是唯一的两个原始布尔值。但是更复杂的布尔表达式可以使用逻辑运算符表示。

此外，Groovy 有特殊规则（通常称为 Groovy Truth），用于将非布尔对象强制转换为布尔值。
